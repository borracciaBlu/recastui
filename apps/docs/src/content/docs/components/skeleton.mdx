# Skeleton

Skeleton is used to display the loading state of some component.

<CodeBlock code={ skeletonSnippets.import } client:visible/>

## Usage
<Playground code={ skeletonSnippets.usage } client:visible/>

Or to wrap another component to take the same height and width.
<Playground code={ skeletonSnippets.wrap } client:visible/>

## Height and Width
You can change the height and/or width of the Skeleton component by using the `h` and `w` props.  The `h` and `w` props accept TailwindCSS height and width values.  You can also use the `full` value to set the height or width to 100%.
Default height is `4` and default width is `full`.
<Playground code={ skeletonSnippets.size } client:visible/>

## Circle
You can set the Skeleton component to have a circular shape by using the `circle` prop.
<Playground code={ skeletonSnippets.circle } client:visible/>

## Loaded
You can set the Skeleton component to be in the loaded state by using the `isLoaded` prop.
<Playground code={ skeletonSnippets.loaded } client:visible/>
