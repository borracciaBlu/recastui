# Avatar

The Avatar component is used to represent user, and displays the profile picture, initials or fallback icon.

## Variants
The Avatar component has 3 *variants* namely *filled*, *outline* & *subtle*. Default is *filled*
<Playground code={ avatarSnippets.variants } client:visible/>

## Rounded
Use the *rounded* prop add roundedness to the Avatar component. There are 5 rounded options: *none*, *sm*, *md*, *lg*, *full*. Default is *none*
<Playground code={ avatarSnippets.rounded } client:visible/>

## Size
Use the *size* prop to change the size of the Avatar component. There are 7 size options: *xxs*, *xs*, *sm*, *md*, *lg*, *xl* & *xxl*. Default is *md*
